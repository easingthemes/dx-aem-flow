## LinkedIn Ads

### Ad Format: Thought Leader Ads (Recommended)
Promote your best-performing personal posts as sponsored content. Highest trust + paid reach.

### Targeting

| Parameter | Values |
|-----------|--------|
| **Job Titles** | Software Engineer, Senior Developer, DevOps Engineer, Engineering Manager, CTO, VP Engineering, Technical Lead, Solutions Architect |
| **Skills** | Azure DevOps, Adobe Experience Manager, CI/CD, GitHub, Jira, Agile, DevOps |
| **Company Size** | 200-10,000+ employees |
| **Industries** | Technology, Financial Services, Healthcare, Retail, Media |
| **Seniority** | Senior, Manager, Director, VP, C-suite |
| **Geography** | US, UK, EU, Australia (start with US + UK) |

### Campaign Structure

**Campaign 1: Awareness (Top of Funnel)**
- Objective: Brand awareness
- Format: Thought Leader Ads (promoted personal posts)
- Budget: $50-100/day
- Content: Hot takes, educational carousels, building-in-public updates
- KPI: Impressions, engagement rate, profile visits

**Campaign 2: Consideration (Mid Funnel)**
- Objective: Website visits
- Format: Sponsored Content (articles, demo links)
- Budget: $50-100/day
- Content: "How to automate your sprint with AI" → link to docs/blog
- KPI: CTR, docs page visits, time on site

**Campaign 3: Conversion (Bottom Funnel)**
- Objective: GitHub stars, installs
- Format: Document Ads (carousels), Direct Sponsored Content
- Budget: $30-50/day
- Content: Quick-start guide, installation walkthrough
- KPI: GitHub star conversions, plugin installs

### Budget Breakdown

| Phase | Monthly Budget | Duration |
|-------|---------------|----------|
| Testing | $1,500-3,000 | Months 1-2 |
| Optimization | $3,000-5,000 | Months 3-4 |
| Scaling | $5,000-8,000 | Months 5+ |

## Tracking & Attribution

- Use UTM parameters on ALL paid links: `?utm_source=linkedin&utm_medium=paid&utm_campaign=awareness_q2`
- Track: GitHub star conversions, docs page visits, plugin install pages
- Set up Google Analytics goals for: GitHub clicks, install page visits, docs engagement
- Review weekly, optimize biweekly, report monthly
