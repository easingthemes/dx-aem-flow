# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In short: be respectful, constructive, and professional in all interactions.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainer. All complaints will be reviewed and investigated.
